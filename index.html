How It Works

1. Fetch Data: The app starts by fetching today's (or tomorrow's, if it's late) games and the latest odds from The Odds API.

2. Calculate Line Movement Momentum: For each game, the app makes a second API call to get the odds from **6 hours ago**. It compares these opening odds to the current odds to calculate the true line movement.

3. Adjust Probability: The initial probability for each team is adjusted based on this line movement. A line moving in a team's favor indicates positive market momentum. This is reflected in the initial position of the slider and the trend arrow (⬆️ or ⬇️).

4. Calculate Expected Value (EV): As you adjust the "True Probability" slider, the app instantly calculates the Expected Value for every available bet using the following formula:

EV = (Your Probability × (Decimal Odds - 1)) - (1 - Your Probability)

A Positive EV (+EV) indicates a bet that is mathematically profitable in the long run, based on your probability estimate.

5. Calculate Kelly Criterion Bet Size: When you enter a bankroll and click "Build Kelly Bets", the app calculates the optimal bet size for the highest +EV opportunity in each qualifying game. This is done to maximize long-term bankroll growth.

Bet Size = Bankroll × (Edge / (Decimal Odds - 1))
